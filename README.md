# Local Area Network (LAN) Simulation
A visual simulation of data flow in a Star Topology using Cisco Packet Tracer.

## 🚀 Features
* Topology: Star Network with 20+ nodes.
* Addressing: Static IPv4 configuration.
* Testing: ICMP (Ping) connectivity across all devices.
